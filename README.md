\# VLIP Platform - Vehicle Lifecycle Integration Platform



\## Features



\### 🔐 Authentication

\- JWT-based authentication

\- Role-based access (Admin, Fleet Manager, Driver)



\### 🚗 Vehicle Management

\- Add, edit, delete vehicles

\- Automatic digital certificates

\- Blockchain-verified ownership



\### 📜 Digital Certificates

\- SHA-256 blockchain hashing

\- Public certificate verification

\- QR code verification



\### 🔮 Predictive Maintenance

\- AI-powered health scoring

\- Component health analysis

\- Maintenance scheduling with cost estimates

\- Personalized recommendations



\### 📊 Dashboard

\- Fleet health trends

\- Component health distribution

\- Recent vehicles

\- Real-time stats



\### 📄 Reports

\- PDF health reports

\- Maintenance history exports

\- Certificate documentation



\### 👥 User Management (Admin)

\- Create users with roles

\- Assign/change roles

\- Delete users
